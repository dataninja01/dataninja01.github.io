Bankruptcy indicator

Indicates the likelihood of an individual filing for bankruptcy. The scores can be used in combination with FICO scores to “sharpen” a lender’s risk assessment for for non-traditional or ‘thin file’ consumers.

  • Libraries used - TensorFlow, Scikit-Learn, FastAPI
  • Training data - Trended bureau attributes, Clear Credit Attributes Features and Alternate data.
  • Ouput - Bankruptcy score.

Media Mix Model

Determining how much each marketing channel is contributing to sales/conversions(attribution). A Marketing Mix Model (MMM) is a technique to estimate the impact of various marketing channels.

  • Libraries used - Pystan, ChannelAttribution
  • Training data - Media spend, Impressions, Web Traffic, Conversions.
  • Ouput - Overall ROAS = total media contribution / total media spending Weekly ROAS = weekly media contribution / weekly media spending.

Propensity to Litigation

Time to event model is a class of statistical methods for studying the occurrence and timing of events. It models factors that influence the time. The hazard score gives the actual probability of going to Litigation for the lifetime of the account.

  • Libraries used - SAS: Survival(phreg)
  • Training data - Account profile metrics, transactional data, Consumer profile.
  • Ouput - Probility in time to litigate.
